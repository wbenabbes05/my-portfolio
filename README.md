# My Portfolio

This is my personal portfolio website built with **React.js** and **Tailwind CSS**. It showcases:

- My **frontend projects** with responsive design and modern UI.
- **Contact section** with email, phone, social links, and download CV button.
- Clean, reusable **React components** and optimized **Tailwind CSS** layouts.
- A professional design suitable for recruiters and clients.

## Features

- Responsive layout for mobile, tablet, and desktop.
- Animated hover effects for buttons and cards.
- Modern design inspired by 2026 frontend standards.
- Easy to customize and extend for future projects.

## Technologies Used

- React.js
- Tailwind CSS
- React Icons
- Git & GitHub for version control

You can deploy this on **GitHub Pages**, **Vercel**, or any hosting service.
